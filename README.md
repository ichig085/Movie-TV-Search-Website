# Interactive Movie & TV Search

## Overview

The Interactive Movie & TV Search is a web application that allows users to search for movies, TV shows, and actors/actresses. It provides a user-friendly interface for exploring information about the entertainment industry.

## Features

- Search for movies, TV shows, actors, and actresses.
- View detailed information about movies and TV shows, including ratings, plot summaries, and cast details.
- Discover popular and top-rated movies and TV shows.
- Responsive design for seamless access on various devices.

## Technologies Used

- HTML
- CSS
- JavaScript
- jQuery
- [The Movie Database (TMDb) API](https://www.themoviedb.org/documentation/api)
- Deployed on [Amazon Web Services (AWS)](https://aws.amazon.com/)

## How to Use

1. Visit the [Interactive Movie & TV Search Website](https://example.com).
2. Enter your search query in the search bar.
3. Explore the search results and click on items for more details.

## Project Structure

- `index.html`: The main HTML file for the website.
- `styles.css`: Custom CSS styles for the website.
- `script.js`: JavaScript/jQuery code for the website functionality.
